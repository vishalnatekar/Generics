Java Testing Materials Project

This project can be built on the commandline by running `./gradlew` on linux or Mac OS or `gradlew` on windows.

It can also be imported into a Java IDE such as Intellij, Eclipse or Netbeans:

 * Intellij (recommended and easiest) - https://www.jetbrains.com/help/idea/getting-started-with-gradle.html
 * Eclipse - https://www.vogella.com/tutorials/EclipseGradle/article.html
